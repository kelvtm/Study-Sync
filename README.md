# Study Sync Project

This is a Node.js backend project using Express and MongoDB.

## How to run

1. Install dependencies:

   ```
   npm install
   ```

2. Start the server:

   ```
   npm start
   ```

3. Visit the backend:
   - [http://localhost:5000/](http://localhost:5000/)
   - [http://localhost:5000/login-page](http://localhost:5000/login-page)

## Features

- User login
- MongoDB integration
